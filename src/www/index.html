Getting Started

A jQTouch app consists of one html page. Everything that is going to show in the app must be inside of a div with the id "jqt"

				<div id="jqt"></div>

Each "page" is just a div inside of that div.

Back

Each time you go to a new page, the page you were on is added to a history. Any "Back" buttons will simply take you one step back in the page history. It's easy to think of pages as being in some sort of hierarchy because of the way that the lists make it feel like you are drilling down. You might expect the back button to take you up one step in the hierarchy. They don't. They just take you back one step in the history.

To see this in action, use the navigation at the bottom of this page and page 5 to go back and forth between this page and page 5 a few times and then click on the "back" button several times.

Toolbar Overview

Toolbars are completely optional, but highly recommended for most pages since they take the place of the browser bar usually and are expected to supply the "back button" functionality.

Toolbar Titles

There isn't much room on the toolbar for titles. You only get 150 pixels and anything after that is dynamically cut of with ellipses

Toolbar Buttons

Adding buttons to the toolbar is a little bit mystifying. If you simply add a link to the page (anywhere) with the class "button" it will throw a non-functioning button up in the toolbar (the button will just take you back to the homepage no matter what). I'll add one right here as example, but you won't see it here, it'll be up in the toolbar!

JSONP

Back

Overview

JSONP allows you to make cross-domain requests as long as the server you are requesting from will respond with JSON

Example

Do JSONP

The following was loaded using JSONP:

    The Code

    <p><a href="#" id="tryJSONP" class="greenButton">Do JSONP</a></p>
<p class="hidden">The following was loaded using JSONP:</p>
<ul id="jsonpTarget" class="plastic hidden"></p>
<script type="text/javascript">
	// http://develop.github.com/p/repo.html
	GitHubAPI = {};

	GitHubAPI.Repos = function(username, callback){
		requestURL = "http://github.com/api/v2/json/repos/show/" + username + "?callback=?";
		jQuery.getJSON(requestURL, function(json, status){
			callback(json.repositories, status);
		});
	};

	jQuery("#tryJSONP").bind("click", function(e){
		e.preventDefault();
		jQuery('#jsonp :hidden').show();
		jQuery('#jsonpTarget').empty().html('<li>loading...</li>');

		GitHubAPI.Repos("cmcculloh", function(json, status){
			var content = [];
			jQuery.each(json, function(i){
				content.push('<li class="project">');
				content.push(this['name']);
				content.push('</li>');
			});
			jQuery("#jsonpTarget").html(content.join(''));
		});
	});
</script>

Overview

Yahoo Pipes can digest any URL and spit out contents you specify (to a certain extent) as JSON

Example

Do Yahoo Pipe

The following was loaded using JSONP from Yahoo pipes:

    The Code

    <p><a href="#" id="tryYahooPipes" class="greenButton">Do Yahoo Pipe</a></p>
<ul id="yahooPipesTarget" class="rounded hidden"></ul>

<script type="text/javascript">
	yahooPipes = {};

	yahooPipes.getDeals = function(callback){
		requestURL = "http://pipes.yahoo.com/pipes/pipe.run?_id=5c2b38b0d1c8cc3a2439662af944d212&_render=json&_callback=?";
		jQuery.getJSON(requestURL, function(json, status){
			console.dir(json);
			callback(json.value.items, status);
		});
	};

	jQuery("#tryYahooPipes").bind("click", function(e){
		e.preventDefault();
		jQuery('#yahooPipes :hidden').show();
		jQuery('#yahooPipesTarget').empty().html('<li>loading...</li>');

		yahooPipes.getDeals(function(json, status){
			var theHTML, $titles;

			theHTML = [];
			contentHTML = json[0].content;

			//sanitize. Remove fancy quotes and make lower case.
			contentHTML = jQuery.replaceByCharCode(contentHTML);
			contentHTML = contentHTML.toLowerCase();
			contentHTML = '<div>' + contentHTML + '</div>';

			$deals = jQuery(contentHTML).find('div.dd2-sub-wrap');

			$deals.each(function(i, deal){
				var title, price, link;

				$deal = jQuery(deal);
				title = $deal.find("span.title").html();
				price = $deal.find("div.dd2-todaysprice").html().replace("today's price: ", "");
				link = $deal.find("a").attr('href');

				theHTML.push('<li class="arrow">'
							, '<a href="' , link , '">'
							, title
							, '</a>'
							, '<small class="counter">'
							, price
							, '</small>'
							, '</li>');
			});
			jQuery("#yahooPipesTarget").html(theHTML.join(''));
		});
	});

	;(function($){
		jQuery.replaceByCharCode=function(n){
			var split_n = n.split('');
			for(var i=0, ii=split_n.length;i<ii;i++){
				var charCode = split_n[i].charCodeAt(0);
				if(jQuery.replaceByCharCode.replaceBy[charCode]){
					console.log('found one!');
					split_n[i] = jQuery.replaceByCharCode.replaceBy[charCode];
				}
			}

			return split_n.join('');
		};

		jQuery.replaceByCharCode.replaceBy = {
			8217: "'",
			146: "'",
			151: "'"
		};
	})(jQuery);
</script>
